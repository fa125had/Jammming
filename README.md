# Codecademy Jammming Project

## Project Goals:
* You can search Spotify's database for your favorite songs,
* add them to a playlist, 
* name the playlist, 
* and save it to your Spotify account. 
* Update: Dec 10 2022:
    * Load | modify | update existing playlists  

## Built With:
* React
* HTML
* SCSS
* Surge
